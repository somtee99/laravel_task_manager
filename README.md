# task_manager

## Project setup
```
composer install
```

### Set up enivoronment for development
- Run to copy env example
```
cp .env.example .env
```
- Run local server
- Create database on local server
- Set mysql database credentials in .env

### Run project
```
php artisan serve
```

### (OPTIONAL) Seed database
- Run to set some default data 
```
php artisan db:seed
```
